This template should help get you started developing with Vue 3 in Vite. The template uses Vue 3 `<script setup>` SFCs, check out the [script setup docs](https://v3.vuejs.org/api/sfc-script-setup.html#sfc-script-setup) to learn more.

Learn more about IDE Support for Vue in the [Vue Docs Scaling up Guide](https://vuejs.org/guide/scaling-up/tooling.html#ide-support).

## 作業回答

### 問題1：問了 AI 哪個問題來搞懂它的程式碼？
我問了AI很多關於Vue程式碼的問題，包括：
- ref的運作原理是什麼？（魔法盒子比喻）
- input的v-model和placeholder是什麼？（魔法繩子和提示牌）
- button的@click意思？（耳朵聽點擊）
- searchUser函數的寫法？（小助手）
- try、response、data的解釋？（安全網、信封、內容）
- div v-if的整塊？（隱形門）
- :src什麼時候需要加冒號？（變數動起來）

這些問題幫助我理解程式碼的每個部分。

### 問題2：AI 給了你什麼「比喻」讓你聽懂了？
AI用很多生活比喻解釋，讓我這個初學者聽懂：
- **ref**：魔法盒子，會通知更新。
- **v-model**：魔法繩子，讓輸入框和變數同步。
- **@click**：耳朵，聽到點擊就呼叫函數。
- **searchUser**：小助手，按鈕按下去時做事。
- **try-catch**：安全網，網路錯時不會崩潰。
- **response**：信封，GitHub的回應。
- **data**：打開信封的內容，用戶資料。
- **v-if**：隱形門，有資料才打開。
- **:src**：冒號讓變數動起來，顯示動態圖片。

這些比喻讓複雜的程式碼變得簡單易懂！
